## Instructions

- You can start the dev server using the following commands:

```sh
# Install tools
asdf install
corepack enable
asdf reshim

# Install dependencies
pnpm install

# Start the dev server
pnpm dev
```

- Open `http://localhost:4321` in your browser to view the website.
